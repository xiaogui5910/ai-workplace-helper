## 二、产品定位

### 1. 核心定位
一款**基于 AI 的职场高情商话术实时生成工具**，用户输入具体情境描述，AI 秒级生成得体、自然、不卑不亢的回复话术，支持多种语气风格切换，帮助职场人轻松应对各类敏感沟通场景。

### 2. 核心价值
- **真 AI 驱动**：不是死板的模板，而是根据用户的具体情境实时生成个性化话术
- **零门槛使用**：打开即用，描述情境即可获得话术，无需注册登录
- **多语气切换**：同一场景可切换温和/坚定/简洁/幽默等多种风格
- **全岗位通用**：覆盖职场高频敏感沟通场景，任何人都能用

### 3. 差异化亮点
- **AI 实时生成 vs 静态模板**：市面上大多是预设话术库，我们是真正的 AI 个性化生成
- **情境感知**：用户描述越详细，生成的话术越贴合实际
- **互动参与感强**：用户可投稿场景、投票最受欢迎话术、分享生成结果
- **轻量工具形态**：单页面应用，打开即用，传播成本极低

### 4.2 功能详述

#### F1：场景快捷选择
- **6 个预设场景按钮**：谈薪、离职、拒绝、汇报、道歉、HR 沟通
- **2 个扩展场景**：跨部门协作、绩效面谈
- **1 个自定义入口**：用户可输入任意职场场景
- 点击场景后，输入框自动填充该场景的引导提示词（placeholder），用户可修改补充

#### F2：情境输入
- 多行文本输入框，用户描述具体情境
- 每个预设场景提供 2-3 个示例情境作为引导（点击可快速填入）
- 输入框下方实时显示字数提示（建议 20-200 字）

#### F3：AI 话术生成（核心功能）
- 点击「生成话术」按钮或按 Enter 触发
- 调用 AI API（豆包/通义千问）实时生成
- **打字机效果**：逐字展示生成结果，增强体验感
- 生成时显示 loading 动画（如"AI 正在为你组织语言..."）
- 每次生成附带一条温馨提示（如"💡 话术仅供参考，可根据实际情况调整"）

#### F4：语气风格切换
- 4 种语气可选，默认「温和」：
  - 🌸 **温和版**：委婉得体，适合对上级/长辈
  - 💪 **坚定版**：有理有据，适合维护自身权益
  - ⚡ **简洁版**：直奔主题，适合高效沟通
  - 😄 **幽默版**：轻松化解尴尬，适合平级同事
- 切换语气后自动重新生成（或从已生成的多个版本中切换）

#### F5：操作功能
- **一键复制**：复制到剪贴板，显示"已复制 ✓"反馈
- **重新生成**：同一情境重新生成不同话术
- **收藏**：使用 localStorage 保存喜欢的话术（收藏列表可查看）
- **分享**：生成分享文案/图片（可选，MVP 后迭代）

#### F6：互动功能（提升参与感和投票）
- **热门场景排行**：使用 localStorage 记录各场景使用次数，展示 TOP3
- **场景征集**：用户提交想要的新场景，存储在 localStorage
- **话术点赞**：每条生成的话术可点赞，展示点赞数
- **每日一句**：每次打开随机展示一条精选职场金句

## 五、技术方案

### 5.1 技术架构

```
前端（HTML + CSS + JS）
    │
    ├── UI 层：页面渲染、交互逻辑
    ├── 数据层：localStorage（收藏、历史、计数）
    └── AI 层：调用大模型 API 生成话术
```

### 5.2 AI API 选型

| 方案 | 免费额度 | 推荐度 | 说明 |
|------|---------|--------|------|
| **豆包（字节跳动）** | 每月 500 万 tokens 免费 | ⭐⭐⭐⭐⭐ | 额度最大，国内访问快，注册火山引擎即可 |
| 通义千问（阿里） | 每日 2000 次免费调用 | ⭐⭐⭐⭐ | 额度充足，通过阿里云百炼平台调用 |
| Kimi（月之暗面） | 注册赠 ~800 万 tokens | ⭐⭐⭐⭐ | 支持超长上下文，适合复杂情境 |

**推荐方案**：优先使用**豆包 API**（免费额度最大，每月 500 万 tokens，足够比赛期间使用）。

### 5.3 API 调用设计

```
用户输入情境描述
    ↓
前端构造 Prompt（含场景 + 情境 + 语气风格 + 约束条件）
    ↓
调用 AI API（fetch 请求）
    ↓
流式返回 / JSON 返回
    ↓
打字机效果展示结果
```

**Prompt 模板设计**：
```
你是一位职场沟通专家，擅长用高情商的方式表达。
当前场景：{scene}
用户情境：{situation}
期望语气：{tone}

请生成一段 50-150 字的职场话术，要求：
1. 语气{tone}，符合职场礼仪
2. 表达清晰自然，不生硬
3. 直接给出话术内容，不需要额外解释
```

### 5.4 API Key 安全
- API Key 硬编码在前端 JS 中（比赛作品，非商业项目可接受）
- 或通过简单的 Base64 编码增加一点"安全性"（防君子不防小人）
- 在底部说明"本工具使用 XX AI 提供 AI 能力"

### 5.5 降级方案
- 如果 API 调用失败（网络问题/额度用完），自动切换到**预设话术库**（每个场景准备 3-5 条高质量预设话术）
- 确保工具在任何情况下都能使用

## 附录 A：Prompt 工程规范

### 系统角色设定
```
你是一位拥有 10 年经验的职场沟通教练，精通各类职场场景的高情商表达。
你的风格是：专业但不生硬，温暖但不谄媚，坚定但不冒犯。
```

### 生成约束
1. 话术长度控制在 50-200 字
2. 不使用过于书面化的表达，保持口语化
3. 不使用emoji（除非用户选择幽默风格）
4. 直接输出话术内容，不添加"以下是话术"等前缀
5. 如涉及具体数据/时间，用 XX 占位，让用户自行替换

### 语气风格定义
- **温和**：多用"理解""建议""可以吗"等柔性词汇，适合对上级
- **坚定**：多用"需要""建议""必须"等明确词汇，适合维护权益
- **简洁**：控制在 50 字以内，直奔主题
- **幽默**：适当加入轻松元素，但保持职场得体

## 附录 B：预设话术库（降级方案）

每个场景准备 3-5 条高质量预设话术，当 API 不可用时随机展示。
（完整话术库在开发阶段由 AI 批量生成并人工审核）
