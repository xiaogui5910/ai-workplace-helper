## 3. BUG 发现 & 修复清单

| # | 严重程度 | 问题 | 修复方案 |

| BUG#3/#18 | 🟡中 | 取消后按钮未恢复 | cancel()恢复按钮 |

| BUG#19 | 🟡中 | 离线消耗限流 | 移除increment |
| BUG#1 | 🟡中 | 限流onclick未恢复 | requestId解决 |
| BUG#6 | 🟡中 | 超时场景不匹配 | 场景快照解决 |
| BUG#5 | 🟢低 | StatsModule null | null守卫 |
| BUG#7 | 🟢低 | 中间💭未闭合 | indexOf截断 |
| BUG#8 | 🟢低 | 敏感信息泄露 | 通用消息 |
