# 🎯 AI 职场嘴替工具箱

> 告别尴尬，AI 帮你说话 — 一键生成高情商职场话术

## 📖 项目简介

AI 职场嘴替工具箱是一款面向职场人士的 AI 话术生成工具。选择沟通场景、描述你的情境、选择语气风格，AI 即刻为你生成专业得体的高情商话术。覆盖谈薪、离职、拒绝不合理需求、向上汇报等 8 大高频职场场景，支持 4 种语气风格切换。

## ✨ 核心功能

### 🎯 智能话术生成
- **8 大预置场景**：谈薪沟通、离职沟通、拒绝不合理需求、向上汇报、职场道歉、HR/猎头沟通、跨部门协作、绩效面谈
- **自定义场景**：支持用户输入任意场景，AI 自动适配
- **4 种语气风格**：温和、坚定、简洁、幽默
- **流式输出**：AI 话术逐字展示，体验流畅

### 📋 一键操作
- **示例一键填入**：每个场景提供 3 个典型示例，点击即可填入
- **一键复制**：生成的话术一键复制到剪贴板
- **重新生成**：不满意可随时重新生成

### 🛡️ 降级保障
- **离线话术**：网络异常或 API 超时时，自动切换到预设话术（打字机效果展示）
- **网络检测**：实时监听网络状态，断网时直接走降级（不等待超时）
- **错误码映射**：7 种 API 错误码自动映射为友好提示

### 📊 互动功能
- **🔥 热门排行**：TOP3 高频使用场景排行
- **👍 话术点赞**：支持点赞/取消，数据本地持久化
- **⏱️ 每日限流**：每日 20 次免费使用，剩余次数实时显示
- **💡 场景征集**：用户可提交想要的新场景建议
- **❌ 取消生成**：生成过程中可随时取消

### 📱 响应式设计
- **PC 端**：居中 640px 宽度，场景按钮 4 列布局
- **移动端**：自适应 2 列布局，触控友好

## 🚀 快速开始

### 在线使用
直接打开 `index.html` 即可使用（需联网调用 AI API）。

### 本地开发
```bash
# 方式一：直接打开
open index.html

# 方式二：本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

### API 配置
API Key 已内置（Base64 编码存储），可直接使用。如需替换：

1. 注册 [MiniMax 开放平台](https://platform.minimaxi.com/)
2. 获取 API Key
3. Base64 编码：`echo -n '你的API Key' | base64`
4. 替换 `index.html` 中 `API_CONFIG.encodedApiKey` 的值

## 🔒 安全性

- **XSS 防护**：全部使用 `textContent`，无 `innerHTML` 拼接用户数据
- **API Key**：Base64 编码存储（比赛作品可接受，生产环境需后端代理）
- **输入校验**：场景名长度限制（30字）、建议内容限制（100字）、情境内容限制（500字）
- **Prompt Injection**：System Prompt 明确指示 AI 忽略用户输入中的指令性内容
